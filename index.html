Automatic Pet Feeder

With the help of Arduino, I built a pet feeding IoT device that enables the user to feed the pet in two ways. First, if the auto mode is enabled the device would detect the pet movement nearby with the use of an ultrasound sensor, consequently activating the step motor to release food. Second, the user can order food release over the phone with a blutooth module.
